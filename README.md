Цей проєкт - вебдодаток на Flask, створений для автоматичного збору та порівняння статистики. Замість того, щоб шукати дані про кожного користувача вручну, програма сама звертається до API й підтягує всю потрібну інфу в реальному часі. Головна ідея полягає в тому, щоб вивести порівняння двох об’єктів на один екран у зручному вигляді.

Програма працює поетапно: спочатку знаходить профіль за ID, перевіряє його доступність і завантажує історію останніх подій. Важлива деталь у тому, що система не просто бере загальний список, а робить окремі запити для кожного пункту. Це дозволяє дістати глибші цифри, як-от показники золота чи досвіду за хвилину, які зазвичай приховані. Також числові ID автоматично замінюються на назви та іконки, щоб на таблицю було легше дивитися.

Щоб усе працювало автоматично, у коді прописана логіка, яка підсумовує результати й видає середні показники - наприклад, вінрейт чи середній рахунок. Оскільки запитів до сервера йде багато, у програму додано обробку помилок і невеликі затримки, щоб сервіс не заблокували за спам. У підсумку виходить інструмент, куди достатньо ввести два ідентифікатори, а система сама все знайде, порахує і видасть готову порівняльну таблицю.
